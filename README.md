# venti1112.github.io

个人网站项目，包含以下功能：

## 网站功能

- 个人主页展示
- RustDesk远程桌面服务器

## RustDesk服务器

提供自托管的RustDesk远程桌面服务器配置连接代码，可用于远程访问和控制计算机。

## 相关链接

- [RustDesk GitHub仓库](https://github.com/rustdesk/rustdesk/)
- [RustDesk官方网站](https://rustdesk.com/zh-cn/)

## 使用说明

1. 下载对应平台的客户端
2. 在客户端中输入服务器连接代码
3. 开始使用远程桌面服务
